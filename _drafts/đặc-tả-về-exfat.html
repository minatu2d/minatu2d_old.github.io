Ngày 27/08/2019 vừa rồi, Microsoft đã công bố đặc tả về hệ thống file exFAT. Cùng với các hoạt động hỗ trợ Linux mạnh mẽ gần đây, Microsoft sẽ đưa exFAT vào nhân Linux.

Mình có quan tâm xem exFAT có gì nổi so với các Filesystem khác.
Mình dự định sẽ dịch đặc tả của nó tại đây:

Đặc tả hệ thống file exFAT

1. Giới thiệu

2. Cấu trúc Volumne (Ổ đĩa logic)

3. Vùng Boot chính và Boot backup

4. Vùng bảng FAT
